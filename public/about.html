The Bot

This system is a Discord bot that listens for commands in a Discord server. It allows users to store and retrieve important dates and times of various events. This website utilizes the same database to serve as an alternative, more streamlined, user interface for server administrators to use. This bot was developed for the Indiana Fighting Game Discord server, as they required a system for keeping track of events such as tournaments within their community.
